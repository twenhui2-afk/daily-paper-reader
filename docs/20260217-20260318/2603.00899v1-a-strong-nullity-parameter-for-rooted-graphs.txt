# A strong nullity parameter for rooted graphs 

# Aida Abiad ∗† Mary Flagg ‡ H. Tracy Hall §

# Jephian C.-H. Lin ¶ Bryan Shader ‖

Abstract 

The inverse eigenvalue problem of a graph G studies the possible spec-tra of matrices associated with G, including as an important subproblem the possible nullities of such a matrix. Much research in this area to date has focused only on the spectrum of the matrix itself, but there are ap-plications of inverse eigenvalue problems that also involve the interaction between that spectrum and the spectrum of some maximal proper prin-cipal submatrix, or in other words the interlacing spectrum that results from crossing out any one row and the same column. Motivated by this refined information, given a graph G on n vertices with a designated root vertex, we investigate all possible nullity pairs where the first nullity is that of an n × n symmetric matrix associated to G and the second nullity is that of the principal submatrix of size ( n − 1) × (n − 1) that results from deleting the row and column associated to the root vertex. We de-fine a new parameter ξξ (G, i ) for rooted graphs ( G, i ) equipped with the strong nullity interlacing property that coordinates the two values of a nullity pair, and we show that this graph parameter is minor monotone. Moreover, we prove a bifurcation lemma for the strong nullity interlacing property. We use these new tools to characterize the rooted graphs with 

ξξ (G, i ) ≥ s for s ∈ { 0, 1, 2, 3, 4, 5} by finding the minimal minors for each of these families. These families turn out to have strong connections to the minimal minors for ξ(G) ≥ k.

Keywords: Inverse eigenvalue problem, strong nullity interlacing property, strong Arnold property, nullity pair, rooted graph minor

# 1 Introduction 

Let G be a graph on n vertices. The set of matrices associated to G is denoted by S(G) and consists of all n × n real symmetric matrices whose off-diagonal (i, j )-entry is nonzero if and only if {i, j } is an edge of G. Here the diagonal entries can be arbitrary real numbers. The inverse eigenvalue problem of a graph 

G (IEP-G) aims to characterize all possible spectra occurs among matrices in 

S(G). The IEP-G was motivated by vibration theory [9, 13–16, 18, 22] and has strong connections to graph structure through for example the Colin de Verdi` ere type parameters [3,4,10,11]. For more details on the IEP-G, see the monograph [20]. While the spectrum of a matrix A ∈ S (G) has been the main focus of the IEP-G, it is natural to consider simultaneously the spectrum of a maximal proper principal submatrix. Let A(i) be the submatrix of A obtained by remov-ing the i-th row and column. The ( λ, μ )-problem, which studies the eigenvalues of A (denoted by λi) and the eigenvalues of A(i) (denoted by μi), has received a fair amount of attention in the literature, see e.g. [7–9, 13–16, 18, 22]. Recently, the i-nullity pair was introduced as (null( A), null( A(i)) and shown to possess a strong relationship to the rooted graph structure [1]. We say that a matrix A ∈ S (G) has the Strong Arnold Property (or has SAP) if X = O is the only symmetric matrix that satisfies A ◦ X = I ◦ X = O and 

AX = O. Using SAP, Colin de Verdi` ere defined a graph parameter μ(G) and showed that it is minor monotone [10, 11]; that is, μ(G) ≤ μ(H) if G is a minor of H. Here G is a minor of H means G can be obtained from H by a sequence of operations of removing an edge, removing an isolated vertex, and contracting an edge. Motivated by the Colin de Verdi` ere parameter μ(G), the parameter 

ξ(G) has been defined as the maximum nullity among matrices A ∈ S (G) with SAP, and this has been shown to be minor monotone as well [3]. In the same paper, it was shown that ξ(G) ≥ 2 if and only if G contains K3 or K1,3 as a minor, and ξ(G) ≥ 3 if and only if G contains a minor in the T3-family, as shown in Figure 3. In general, if a graph parameter ζ(G) is minor monotone, then a graph G is said to achieve ζ(G) ≥ k as a minimal minor if ζ(G) ≥ k and any proper minor 

F of G has ζ(F ) < k . Thus, one may characterize graphs G with ζ(G) ≥ k by finding all minimal minors, of which there are only finitely many for each k by the Graph Minors Theorem; see, e.g., [12]. A similar idea also works for rooted graphs. A rooted graph (G, i ) is a simple graph G with a vertex i designated as the root. We say ( G, i ) is a rooted minor of ( H, j ) if ( G, i ) is obtained from a rooted graph ( H, j ) by removing an edge, removing a vertex that is not the root, or contracting an edge, where the new vertex given by the contraction becomes the root of the minor if the contracted edge is incident to the original root. A rooted graph parameter ζ(G, i ) is then said to be minor monotone if 

ζ(G, i ) ≤ ζ(H, j ) whenever ( G, i ) is a rooted minor of ( H, j ). By an analogous graph minors theorem for rooted graphs [23], the set of minimal minors for 

ζ(G, i ) ≥ s is also finite for any given value of s.In [1], analogous statements for matrices and its submatrices were introduced 2and investigated. Let ( G, i ) be a rooted graph and let A be a matrix in S(G). We say A has the i-strong nullity interlacing property (or has i-SNIP) if X = O

is the only symmetric matrix that satisfies the identities A ◦ X = I ◦ X = O

and ( AX )( i, :] = O. Here ( AX )( i, :] designates the (non-square) submatrix of 

A obtained by removing the i-th row. We say ( G, i ) allows the nullity pair (k, ℓ ) (with i-SNIP, respectively) if there is a matrix A ∈ S (G) that achieves the i-nullity pair ( k, ℓ ) (with i-SNIP, respectively). It is a result of [1] that this new strong property enforces minor monotonicity: If ( G, i ) allows the nullity pair ( k, ℓ ) with i-SNIP and is a rooted minor of ( H, i ), then ( H, i ) also allows (k, ℓ ). Using this minor monotonicity theorem, the rooted graphs that allow (0 , 0), (0 , 1), (1 , 1), or (1 , 2) with i-SNIP were characterized in [1], in each case in terms of minimal minors. In this paper we introduce a new parameter ξξ (G, i ), which is defined as the maximum k + ℓ for some k ≤ ℓ such that there is a matrix A ∈ S (G) with the 

i-nullity pair ( k, ℓ ), k ≤ ℓ, and i-SNIP. Minor monotonicity can be rephrased as 

ξξ (G, i ) ≤ ξξ (H, j ) whenever ( G, i ) is a rooted minor of ( H, j ). Besides minor monotonicity, we also prove a bifurcation lemma for ξξ (G, i ), showing that every non-negative sum up to the maximum can be achieved: If k + ℓ ≤ ξξ (G, i ) with 

k ≤ ℓ, then ( G, i ) also allows i-nullity ( k, ℓ ) with i-SNIP. As a result, this new parameter coordinates both values in a nullity pair at once—for example, ( G, i )allows (1 , 2) with i-SNIP if and only if ξξ (G, i ) ≥ 3. In other words, the minimal minor results of [1] can be restated as enumerating all of the minimal minors that characterize ξξ (G, i ) ≥ s for s ∈ { 0, 1, 2, 3}. In the paper at hand, we advance these results by extending the characterization of ξξ (G, i ) ≥ s to s = 4 and s = 5. The minimal minors for ξξ (G, i ) ≥ s for s ∈ { 0, . . . , 5} are shown in Figure 1. A further result of the present work is that for any k ≥ 0, the minimal minors for ξξ (G, i ) ≥ 2k + 1 are obtained from the minimal minors for 

ξξ (G, i ) ≥ 2k by appending a leaf as the new root to the original root. This behavior was previously observed in the minimal minors for ξξ (G, i ) ≥ s in the cases s ∈ { 0, 1} and s ∈ { 2, 3}, but here we show that in fact this holds for any pair s ∈ { 2k, 2k + 1 }.This paper is organized as follows. Section 2 introduces the needed back-ground and notation. Section 3 presents the Bifurcation Lemma and related results, which show that the allowed nullity pair sums are consecutive. In Sec-tion 4 we show that the minimal minors of ξξ (G, i ) ≥ 2k + 1 are obtained from the minimal minors of ξξ (G, i ) ≥ 2k by appending a leaf. In Section 5, we study how nullity pairs interact with a Schur complement construction, which is es-sential for the section that follows. Building upon these results, we characterize the minimal minors of ξξ (G, i ) ≥ 4, 5 in Section 6. Finally, we provide a com-prehensive study on i-SNIP in Section 7, including the proof of the Bifurcation Lemma, the edge bound, and an equivalent null space definition for i-SNIP. The final Section 8 contains some concluding remarks and an open question based on observations arising in this work. 3ξξ (G, i ) ≥ 0 = 0 + 0 (K1, v )

# 2 Preliminaries 

Let A be a real symmetric matrix and i an index. It is known that in every instance, exactly one of the following cases holds (see, e.g., [1, Remark 2.1]): 

• null( A) + 1 = null( A(i)), 

• null( A) = null( A(i)), 

• null( A) − 1 = null( A(i)). Depending on which of these three cases holds, we say the index i is upper ,

neutral , or downer , respectively. A consequence of this characterization is that any nullity pair (null( A), null( A(i)) = ( k, ℓ ) necessarily satisfies |k − ℓ| ≤ 1. On the other hand, we know that the cases of ( k, k ) and ( k + 1 , k ) are equivalent in the following sense. 

Proposition 2.1 (Corollaries 2.3 and 3.10 of [1]) . Let (G, i ) be a rooted graph. Then (G, i ) allows the nullity pair (k, k ) (respectively, with i-SNIP) if and only if (G, i ) allows the nullity pair (k + 1 , k ) (respectively, with i-SNIP). 

Therefore, we focus on the nullity pairs ( k, ℓ ) when k ≤ ℓ and define the parameter ξξ (G, i ) as the largest k + ℓ such that ( G, i ) allows the nullity pair (k, ℓ ) with k ≤ ℓ.4Also, [1, Lemma 3.6] states that if A has i-SNIP, then both A and A(i)have SAP. Therefore, if ( G, i ) allows the nullity pair ( k, ℓ ) with i-SNIP and with 

k ≤ ℓ, then k ≤ ξ(G) and ℓ ≤ ξ(G − i) ≤ ξ(G). This gives ξξ (G, i ) ≤ 2ξ(G), justifying the choice of notation. In fact, i-SNIP can be fully characterized by checking SAP for appropriate matrices. Denote by Ei,i the n × n matrix whose ( i, i )-entry is 1 while all other entries are 0. 

Theorem 2.2 (Theorem 3.9 of [1]) . Let A ∈ Sym n(R) and let i be an index in 

{1, . . . , n }. Then the following characterization holds. 

(a) If i is a downer index of A, then A has i-SNIP if and only if A has SAP. 

(b) If i is a neutral index of A and t is the unique value such that i is a downer index of A + tE i,i , then A has i-SNIP if and only if A + tE i,i has SAP. 

(c) If i is an upper index of A, then A has i-SNIP if and only if A(i) has SAP. 

When A and B are real symmetric matrices and i is an index of A, [1, Proposition 3.5] stated that the direct sum A ⊕ B has i-SNIP if and only if A

has i-SNIP and B is invertible, which leads to Proposition 2.3. This allows us to reserve attention to connected graphs. 

Proposition 2.3. Let G ˙∪ H be the disjoint union of graphs G and H with 

i ∈ V (G). Then 

ξξ (G ˙∪ H, i ) = ξξ (G, i ).

Finally, we introduce notation for submatrices. Let A be a matrix, α a subset of row indices, and β a subset of column indices. Then A[α, β ] stands for the submatrix of A induced on the rows in α and columns in β. In contrast, A(α, β )is the submatrix of A obtained by removing the rows in α and the columns in β.The meanings of A(α, β ] and A[α, β ) are also straightforward. For convenience, we write A[α] = A[α, α ], A(α) = A(α, α ), and A(i) = A({i}). The notation : is used to denote the set of all indices, so A(i, :] is the non-square submatrix of A

obtained by removing only row i of A.

# 3 Bifurcation Lemma 

Since | null( A) − null( A(i)) | ≤ 1 for any matrix A and index i, all nullity pairs (k, ℓ ) that occur are grid points in the first quadrant of the Cartesian plane lying on one of the lines ℓ = k + 1, ℓ = k, or ℓ = k − 1. It is natural to ask the following question: If ( G, i ) allows ( k, ℓ ), does that imply that ( G, i ) allows ( k − 1, ℓ ) or ( k, ℓ − 1)? We answer this question affir-matively (structuring the proof in three cases), and illustrate this in Figure 2. 1. The step in the second case does not require i-SNIP, and also does not necessarily preserve i-SNIP. The third case both requires and preserves i-SNIP. Now we prove the second case. 

Theorem 3.1. For k, ℓ ≥ 0, if a rooted graph (G, i ) allows (k + 1 , ℓ + 1) , then it allows (k, ℓ ).Proof. Let A ∈ S (G) with the i-nullity pair ( k + 1 , ℓ + 1). We first claim that there is a nonzero vector v ∈ ker( A) such that its i-th entry ( v)i is 0. To see this, we consider two cases. First, we consider the case when k + 1 ≥ 2. Since any basis of ker( A) contains at least two vectors, we may obtain a linear combination of them to create a nonzero vector v ∈ ker( A) with ( v)i = 0. Second, we consider the case when k + 1 = 1 and ker( A) = span {v} for some vector v. If ( v)i = 0, then we are done. Otherwise, ( v)i̸ = 0 implies that column 

i of A is a linear combination of other columns. Since A is symmetric, this means null( A(i)) = null( A) − 1 = 0 = ℓ + 1, which contradicts the assumption ℓ ≥ 0. Let v ∈ ker( A) be a nonzero vector with ( v)i = 0. Let j be an index such that ( v)j̸ = 0. Necessarily, j̸ = i. We claim that A′ = A + Ej,j has the i-nullity pair ( k, ℓ ). To see this, we observe that ( v)j̸ = 0 implies that column j of A

is a linear combination of other columns, so null( A(j)) = null( A) − 1 = k. On the other hand, after row and column operations on A′, column j and row j

6are left with a single nonzero entry 1 occupying the ( i, i )-position. Therefore, null( A′) = null( A(j)) = k. On the other hand, let u be the vector obtained from v by removing entry i. Then u ∈ ker( A(i)) is a nonzero vector with (u)j̸ = 0. For the same reason, we have null( A′(i)) = null( A({i, j }) = null( A(i)) − 1 = ℓ. 

For the third case, we need the Bifurcation Lemma (Theorem 3.2), which will be proved in Section 7.

# 8 Concluding remarks 

In this paper, we introduced a new parameter ξξ (G, i ) as a tool to understand the nullity pair problem. We established various theoretical results for i-SNIP, including the Bifurcation Lemma, an edge bound, and a null space definition of i-SNIP. Using these results we were able to characterize the graphs with 

ξξ (G, i ) ≥ s for s ∈ { 4, 5}. All known minimal minors for ξξ (G, i ) ≥ s are summarized in Figure 1. 

Question 8.1. In the known cases of minimal minors for ξξ (G, i ) ≥ 2k, corre-sponding to values k ∈ { 0, 1, 2}, the minimal minors are precisely the minimal minors for ξ(G) ≥ k + 1, rooted at any vertex that is not a cut-vertex. Does this pattern continue for k > 2? We proved that the minimal minors for ξξ (G, i ) ≥ 2k + 1 and the minimal minors for ξξ (G, i ) ≥ 2k can be obtained from each other, reducing the cases to be studied by half. (This implies in particular that any case of an answer to Question 8.1 would also resolve the case ξξ (G, i ) ≥ 2k + 1.)
